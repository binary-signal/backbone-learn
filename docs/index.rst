BackboneLearn: A Framework for Scaling Mixed Integer Optimization Problems with Indicator Variables to High-Dimensional Problems
===============================================================================================================================

Introduction
------------

Welcome to the BackboneLearn documentation. This framework is designed to scale mixed integer optimization problems with indicator variables to high-dimensional problems, providing advanced tools and methodologies for researchers and developers. Here, you will find all the necessary information to get started, understand the core concepts, and effectively use or contribute to BackboneLearn.
